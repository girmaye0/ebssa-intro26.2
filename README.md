# 🐶 Dog Breed Explorer & Portfolio

Welcome to my web development repository! This project features a personal portfolio homepage paired with an interactive _Dog Breed Explorer_ web application that connects to an external REST endpoint to dynamically display dog pictures, temperaments, and characteristics.

## Key Features

- Dual-Layer Layout Navigation: Built a sticky, top-level navigation system that flows smoothly between multi-page layouts and same-page anchor links.
- Asynchronous Integration: Leverages JavaScript Promises (`.then()` chaining) to populate dropdown elements dynamically on page load.
- Smart UI Fallbacks: Uses conditional OR operators (`||`) to cleanly handle missing API values (such as blank breed purposes) with professional default text.
- Separation of Concerns: Isolated configurations like secret API keys into dedicated environment objects away from the core application logic.
